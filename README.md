I'd recommend using LinuxAcademy, QwikLabs, practice exams, and reviewing Google Cloud product docs at MINIMUM.

General tips: focus on the mock case study questions, as those are the trickier ones to answer and will likely take up between 15-20 questions of the exam. The remaining questions will test your knowledge on the platform and demand that you apply critical thinking and an architect's perspective to design cloud native solutions. Topics to be familiar with: picking storage solutions, networking, migrating on-prem to managed cloud solutions, IAM.

Good luck!
